Tutorial de Configuração com Supabase (Gratuito, sem cartão)

PARTE 1: Criar conta no Supabase

1 Acessar o Supabase

  1. Acesse: https://supabase.com/
  2. Clique em "Start your project" (botão verde)
  3. Faça login com GitHub ou Google

2 Criar novo projeto

  1. Clique em "New Project"
  2. Preencha:
    • Name: imperio-loja
    • Database Password: crie uma senha (guarde essa senha!)
    • Region: South America (São Paulo)
  3. Clique em "Create new project"
  4. Aguarde 1-2 minutos enquanto cria

3 Copiar as credenciais

  1. No painel do projeto, vá em "Project Settings" (ícone de engrenagem)
  2. Clique em "API" no menu lateral
  3. Copie:
    • Project URL (ex: https://xxxxx.supabase.co)
    • anon public key (a chave grande)

⚠️ IMPORTANTE - COLE NO ARQUIVO:

Abra o arquivo supabase-config.js e cole as informações:

const SUPABASE_URL = 'https://xxxxx.supabase.co';    // ← Cole sua URL aqui
const SUPABASE_ANON_KEY = 'eyJhbGci...';             // ← Cole sua key aqui

PARTE 2: Criar o Banco de Dados

4 Criar tabela de produtos

  1. No menu lateral, clique em "SQL Editor"
  2. Clique em "New query"
  3. Cole o código abaixo e clique em "Run" (ou Ctrl+Enter):
-- Criar tabela de produtos
CREATE TABLE products (
    id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
    name TEXT NOT NULL,
    price DECIMAL(10,2) NOT NULL,
    category TEXT NOT NULL,
    sizes TEXT[] DEFAULT '{}',
    image_url TEXT,
    archived BOOLEAN DEFAULT false,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

-- Permitir acesso público para leitura
ALTER TABLE products ENABLE ROW LEVEL SECURITY;

CREATE POLICY "Permitir leitura pública" ON products
    FOR SELECT USING (true);

CREATE POLICY "Permitir inserção pública" ON products
    FOR INSERT WITH CHECK (true);

CREATE POLICY "Permitir atualização pública" ON products
    FOR UPDATE USING (true);

CREATE POLICY "Permitir exclusão pública" ON products
    FOR DELETE USING (true);
✅ Sucesso! Se aparecer "Success. No rows returned" está correto!

5 Criar Storage para fotos

  1. No menu lateral, clique em "Storage"
  2. Clique em "New bucket"
  3. Nome: products
  4. IMPORTANTE: Marque a opção "Public bucket"
  5. Clique em "Create bucket"

6 Configurar permissões do Storage

  1. Clique no bucket "products" que você criou
  2. Clique na aba "Policies"
  3. Clique em "New policy"
  4. Selecione "For full customization"
  5. Preencha:
    • Policy name: Permitir tudo
    • Allowed operations: Marque TODAS (SELECT, INSERT, UPDATE, DELETE)
    • Target roles: anon
  6. Em "WITH CHECK expression" digite: true
  7. Em "USING expression" digite: true
  8. Clique em "Review" e depois "Save policy"
✅ Pronto! O Supabase está configurado. Agora vamos hospedar o site.

PARTE 3: Hospedar o Site (Vercel - Grátis)

7 Criar conta no GitHub

  1. Acesse: https://github.com/
  2. Clique em "Sign up" e crie uma conta
  3. Ou faça login se já tiver conta

8 Subir os arquivos

  1. Clique em "New" (botão verde) para criar repositório
  2. Nome: imperio-loja
  3. Marque "Public"
  4. Clique em "Create repository"
  5. Clique em "uploading an existing file"
  6. Arraste TODOS os arquivos da pasta Imperio Roupas (incluindo a pasta Foto)
  7. Clique em "Commit changes"
📁 Arquivos que você deve subir:
  • index.html
  • admin.html
  • style.css
  • app.js
  • admin.js
  • supabase-config.js (já com suas credenciais)
  • Pasta Foto/ (com as imagens)

9 Conectar ao Vercel

  1. Acesse: https://vercel.com/
  2. Clique em "Start Deploying"
  3. Faça login com GitHub
  4. Clique em "Import" no repositório imperio-loja
  5. Clique em "Deploy"
  6. Aguarde 1-2 minutos
✅ Pronto! Você receberá um link tipo: imperio-loja.vercel.app

Como adicionar uma roupa:

  1. Acesse seu-site.vercel.app/admin.html
  2. Digite a senha: imperio2024
  3. Clique em "+ Nova Roupa"
  4. Selecione a Categoria
  5. Clique para selecionar a foto
  6. Preencha nome e preço
  7. Marque os tamanhos disponíveis
  8. Clique em "Salvar"

Alterar a senha do Admin:

  1. Abra o arquivo admin.js
  2. Na linha 7, altere: const ADMIN_PASSWORD = 'imperio2024';
  3. Suba o arquivo no GitHub

Alterar o WhatsApp:

  1. Abra o arquivo app.js
  2. Na linha 7, altere: const WHATSAPP_NUMBER = '5517997539595';
  3. Suba o arquivo no GitHub

Checklist de Configuração

  • Criei conta no Supabase
  • Criei o projeto
  • Copiei URL e anon key para supabase-config.js
  • Criei a tabela products (rodei o SQL)
  • Criei o bucket "products" no Storage
  • Configurei a policy do Storage
  • Criei conta no GitHub
  • Subi os arquivos para o GitHub
  • Conectei ao Vercel
  • Testei a loja
  • Testei o admin
  • Adicionei minha primeira roupa

Dúvidas Frequentes

O site não carrega os produtos:

Verifique se as credenciais estão corretas no arquivo supabase-config.js

Erro ao subir foto:

Verifique se criou o bucket "products" como público e configurou a policy.

Esqueci a senha do admin:

Abra o arquivo admin.js e veja a senha na linha 7.
